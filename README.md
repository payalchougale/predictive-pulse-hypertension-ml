# Predictive Pulse: Harnessing Machine Learning for Blood Pressure Analysis

This project uses Machine Learning to predict different stages of hypertension (high blood pressure) based on clinical parameters and lifestyle factors.

The system analyzes patient data such as age, symptoms, blood pressure levels, and medical history to classify hypertension stages.

Technologies Used:
- Python
- Machine Learning (Scikit-learn)
- Pandas & NumPy
- Matplotlib & Seaborn
- Flask Web Framework

Features:
- Hypertension stage prediction
- Risk assessment
- Clinical recommendations
- Web-based interface for patient data input
